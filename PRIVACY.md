# Política de Privacidad

## Protección de Datos

Este dashboard procesa datos financieros sensibles. La información incluida es confidencial y debe ser tratada como tal.

### Datos Recolectados
- Información de asesores
- Datos de transacciones financieras
- Información de clientes/empresas
- Fechas de pago

### Seguridad
- **Almacenamiento Local**: Los datos se cargan desde archivos locales
- **Conexión Segura**: Usa HTTPS en Streamlit Cloud
- **Sin Servidores Externos**: Los datos no se envían a servidores de terceros
- **Caché Local**: Streamlit almacena en caché solo en el servidor

### Acceso
- Solo usuarios autorizados deben tener acceso a este dashboard
- Considera usar autenticación (Streamlit Cloud soporta SSO)
- No compartas el enlace público en redes sociales

### Cumplimiento Legal
- Asegúrate de cumplir con leyes de protección de datos local
- En Perú: LPDP (Ley de Protección de Datos Personales)
- En otros países: GDPR, CCPA, etc.
